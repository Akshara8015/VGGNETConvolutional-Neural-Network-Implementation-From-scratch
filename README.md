# VGGNET-CNN-Implementation-From-scratch

This repository contains a from-scratch implementation of the VGG-style Convolutional Neural Network, built to understand the core internals of deep CNN architectures without relying on pretrained models.

# Tech Stack

-Python
-NumPy
-PyTorch
-Matplotlib

# Key Learnings

-CNN building blocks (conv, pooling, padding, stride)
-Deep network design principles
-Training dynamics & optimization challenges
- How VGG-style architectures work
